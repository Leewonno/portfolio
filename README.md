# 포트폴리오 프로젝트

## 🌟 프로젝트 소개
이 프로젝트는 `React`와 `TypeScript`를 기반으로 개발된 저의 개인 포트폴리오 웹사이트입니다. 개발자로서의 경험, 활용 가능한 기술 스택, 그리고 지금까지 진행해 온 다양한 프로젝트들을 소개하기 위해 제작되었습니다.

## ✨ 주요 내용
- 자기소개 (About Me): 저의 개발 철학, 경력, 그리고 강점 등을 기술하여 저에 대한 이해를 돕습니다.
- 기술 (Skills): 제가 주로 사용하는 프로그래밍 언어, 프레임워크, 라이브러리 및 도구들을 아이콘과 함께 시각적으로 제공합니다.
- 프로젝트 (Projects): 진행했던 주요 프로젝트들을 개요, 사용 기술, 역할, 학습 내용 등과 함께 보여줍니다.
- 이전 버전 : 푸터(Footer) 하단에 `버전2`, `버전1` 링크를 클릭하면 저의 포트폴리오 페이지 변화 과정을 확인할 수 있습니다.

## 🛠️ 기술 스택
| Category | Technology |
|--------|--------|
| Frontend | React, TypeScript, Styled Components |
| Backend | Firebase Cloud Firestore |
| Deployment | Firebase Hosting |

## 🚀 개발 과정
- 전체 기획 및 개발: 프로젝트의 아이디어 구상부터 설계, 개발, 배포까지 전 과정에 참여하여 웹사이트를 완성했습니다.
- 컴포넌트 기반 개발: React의 컴포넌트 아키텍처를 활용하여 재사용 가능하고 유지보수가 용이한 코드베이스를 구축했습니다.
- 타입 안정성 확보: TypeScript를 도입하여 개발 과정에서 발생할 수 있는 잠재적인 오류를 줄이고 코드의 안정성을 높였습니다.
- 클라우드 기반 배포: Firebase Hosting을 이용하여 빠르고 안정적인 웹사이트 배포 및 관리를 수행했습니다.
